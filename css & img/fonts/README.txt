INSTRUKSI PENGGUNAAN FONT COTORIS
==================================

Font Cotoris adalah font premium dari Canva.

CARA MENAMBAHKAN FONT COTORIS:
1. Unduh font Cotoris dari Canva (jika Anda memiliki akses premium)
2. Simpan file font ke folder ini (css & img/fonts/)
3. Format file yang didukung:
   - Cotoris-Regular.woff2 (prioritas utama)
   - Cotoris-Regular.woff
   - Cotoris-Regular.ttf
   - Cotoris-Bold.woff2 (untuk teks tebal)
   - Cotoris-Bold.woff
   - Cotoris-Bold.ttf

4. Pastikan nama file sesuai dengan yang ada di CSS:
   - Cotoris-Regular.woff2 / .woff / .ttf
   - Cotoris-Bold.woff2 / .woff / .ttf

5. Jika font belum diupload, browser akan menggunakan fallback font (serif)

CATATAN:
- Font Cotoris mungkin memerlukan lisensi dari Canva
- Pastikan Anda memiliki hak untuk menggunakan font ini secara komersial
- Jika tidak memiliki font Cotoris, Anda bisa menggunakan font serif alternatif seperti:
  - Playfair Display
  - Cormorant Garamond
  - Lora
  - Merriweather
